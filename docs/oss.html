Building software in the open has been a lot of fun for a few years and it taught me a lot about collaboration and high-quality code bases. Here are some projects I’m particularly fond of:

Heidi

Heidi (Haskell + tidy) is an experiment in dataframe UX for a strictly-typed functional language. It uses generic programming to turn values from algebraic datatypes into row types (implementing half of an in-memory ORM, so to speak).

sparse-linear-algebra

A very ambitions sparse linalg library, providing linear solvers, factorizations (LU, QR) and eigensolvers. It was very instructive to try my hand at this, and I’m grateful for the early adopters and contributors, but getting this right+performant is a huge task that should be delegated to more established libraries.
